Muchos de los mods que hago, usan un único archivo **.ini** para las configuraciones de estos, llamado **cleo_settings**. Si lo tienes, editelo con un editor de texto con las lineas de abajo. Si no tienes uno, tambien puedes crearlo o utilizar el que viene en el archivo comprimido. Para personalizar este mod con estas líneas: ```ini [MDsVelocimeter]; --> https://matidragon-yt.github.io/page/mods/m ; Km/h - 4.0 ; M/h - 1.0 ; M/s - 0.25 TextAtDisplay=Km/h MultipliedNumber=4.0 samp=1 ``` ###### Parametros 1. `DisplayText` - Contiene el texto que se mostrara arriba del numero del medidor. 2. `MultipliedNumber` - Se usa para hacer mas grande número que se muestra. 3. `samp` - Si desea activarlo en SAMP con un 1, ó NO con un 0. ###### Ejemplos |formato de medida|texto|número| |-| |Kilometros por hora|Km/h|4.0| |Metros por hora _(por defecto)_|M/h|1.0| |Metros por segundo|M/s|0.25| ::: ::: :::l [descargar](./docs/m/MDs-Velocimeter.zip 'button') ::: ::: :::d-none d-md-block col-lg-6 py-4 !yt[](XJ1bve5lulI) :::mx-2 ![](https://img.shields.io/youtube/likes/XJ1bve5lulI?style=for-the-badge) ![](https://img.shields.io/youtube/views/XJ1bve5lulI?style=for-the-badge) ![](https://img.shields.io/youtube/channel/subscribers/UCIqJ7P_fLvULqvmsDagJBjA?style=for-the-badge) ::: **Historial de cambios** v 1.2\n \s\s\sSe agrego personalizacion via archivo ini\n v 1.1\n \s\s\sSe optimiso, y agrego soporte para autos\n v 1.0\n \s\s\sNada especial, solo disponible para motos ::: :::
