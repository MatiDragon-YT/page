### SCMTool

Sabemos que Sanny Builder es un gran IDE, pero trabajar memorizando opcodes puede ser algo desagradable para muchos, por lo que este parche para que tengas otra alternativa para codificar.

  • Una de las cosas que trae este paquete, son los siguientes:
    * Una documentación detallada sobre SB y GTA SA. * Traducción precisa de todo el IDE al Español Latino. * Más de 80 clases, 2000 miembros y 100 enumeraciones. * Ayuda para SAMP Funcs, CLEO Plus, CLEO ImGui, New Opcodes y Shader API. * Unos cuantos temas de colores simples y modernos.

Para ver que Class o Keyword corresponde a cada Opcode, descompile sin codigo fuente.
Si actualizo SCMTool, use antes la directiva {$NOSOURCE} para actualizar automaticamente los comandos. 
{$CLEO}
{$NOSOURCE}
thread 'EXAMPLE'
while true
    0@ = Actor.Create(PedType.Gang2, #FAM1, 1524.823, -2433.53, 13.5547)
    repeat
        wait 0
    until Key.isPressed(VK.R)
    Actor.Release(0@)
end
