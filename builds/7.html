### Opty RenderHook

Se trata del mismo mod, pero optimizado en un solo paquete para una instalacion rapida y fuera de problemas. Ademas esta version consume un -50% de consumo de nuestro equipo.

#### Equipo que se uso ``` Notebook EXO Smart R9 Sistema Operativo Windows 10 Home Procesador Intel® Celeron™ N3060 x64 Potencia de @1.60GHz (2CPUs). ~1.6GHz Intel(R) HD Graphics 4000 2GB de VRAM Compartido 1x4GB RAM SodImm DDR3 Almacenamiento SATA 2.5 de 500GB Pantalla de 14” HD+ Resolución de 1600x900 BacklightLED ```
#### El paquete contiene ``` +GTA SA 1.0 US +CLEO 4.4 +CLEOImGui 1.2 +CLEOPlus 1.0.8 +Mod Loader +SilentPatchSA 1.1 +RenderHook 27/9/2018 (Menu con [F2]) +ReShade 4.3.0 (Menu con [Inicio]) +MemoryFix 1024 +FPS for Cinematics 1.1 ```
Borre "FPSforCinematics.cs" de la carpeta "CLEO" si usas el mod "FPS de Reserva" o viceversa
